# PingPong Pro 🏓

A premium React application built with Vite, featuring a modern design system, glassmorphism, and smooth animations.

## Getting Started

To get the project running locally, follow these steps:

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Tech Stack
- **React 18**
- **Vite** (Next Generation Frontend Tooling)
- **Lucide React** (Beautiful & consistent icons)
- **Vanilla CSS** (Custom glassmorphism & gradients)

## Project Structure
- `src/App.jsx`: Main application component with hero section.
- `src/App.css`: Custom premium styling.
- `src/index.css`: Global base styles and color tokens.
